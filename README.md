# HRMS Lite

## Overview
Lightweight HRMS for employee and attendance management.

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI
- DB: SQLite

## Local Setup
1. Backend: `cd backend`, `pip install -r requirements.txt`, `uvicorn main:app --reload`
2. Frontend: `cd frontend`, `npm i`, `npm run dev`

## Assumptions
- No auth (single admin).
- Simple UI without extras.
